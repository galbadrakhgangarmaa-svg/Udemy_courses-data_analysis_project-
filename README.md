Hello all,

In this project I'm exploring Udemy courses data for data analysis practice. 

This project more focused on answering business questions to help and give idea of what kind of business questions we can answer using the datases and familiar ones.

Since data is in quite good condition, I had no need to do lot of cleaning but only deal with NULL values and data validation. 

For the analysis, I divide my analysis to 5 main section. Each section will focus on different area of data exploration. 

To make data analysis more easy I made data enrichment making the data more easy to analyse and understand. 

Data analysis sections : 
 1. Content analysis can help to understand following business questions:
      1.1 Which course/subject gets the most subscribers?
     1.2 Do courses with more lectures get more students?
     1.3 Does course duration affect course popularity?
     1.4 Are shorter courses or longer courses more popular?
2. Course analysis /help for decision of target audiences/
    2.1 Which level has most subs?
    2.2 Do beginner courses more popular than advanced ones?
    2.3 Which level recieves the highest reviews?
3. Engagement analysis / what makes students more satisfied/
    3.1 Do courses with more lectures get more reviews?
    3.2 Do longer courses generates more reviews?
    3.3 What is the relationship between subs and reviews?
    3.4 Which courses have higt subs but low reviews?
4. Time analysis / help to understand growth trends/
    4.1 How was the number of courses changed over time?
    4.2 Which years had the most successful courses?
    4.3 Are newer courses getting more students?
    4.4 Is the average course duration increasing over time?
6. Identify top 10
    5.1 TOP 10 courses by subs
    5.2 TOP 10 courses by reviews
